# Secrets

## Secrets kubectl

```
kubectl create -f <secret_definition>

kubectl get secrets [<secret_name>]
kubectl get secrets [<secret_name>] -o wide
kubectl describe secrets [<secret_name>]
kubectl explain secrets
```

## Secrets yml

```
apiVersion: v1
kind: Secret
metadata:
  name: newsecret
data:
  password: TEZUckAxbgo=
```

Mount a secret to a pod as a `volume`
```
apiVersion: v1
kind: Pod
metadata:
  name: securityapp
  labels:
    type: security
spec:
  securityContext:
    runAsUser: 1000
  containers:
  - name: busy
    image: busybox
    command:
      - sleep
      - "3600"
    securityContext:
      runAsUser: 2000
      allowPrivilegeEscalation: false
      capabilities:
        add: ["NET_ADMIN", "SYS_TIME"]
    volumeMounts:
    - name: mysql
      mountPath: /mysqlpassword
  volumes:
  - name: mysql
    secret:
      secretName: newsecret
```

See the secret inside the container (pod)
```
kubectl exec -it <pod_name> -- sh

/ $ cat /<pod.spec.containers.volumeMounts.mountPath>/<secrets.data>

/ $ cd /<pod.spec.containers.volumeMounts.mountPath> && ls -la
> password -> ..data/password
```
